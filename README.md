# 🛒 E-Commerce Retail Performance Analysis

## 📌 Project Overview
Analyzed 10,000+ transactional records from an online retail dataset to uncover quarterly revenue trends, top-performing products, and customer buying patterns across multiple countries.

Built interactive dashboards in both **Power BI** and **Tableau** to visualize Q1–Q4 performance and support data-driven business decisions.

---

## 🛠️ Tools Used
- Power BI (Dashboard + DAX)
- Tableau (Quarterly Worksheets)
- Microsoft Excel (Data Source)

---

## 📊 Key Insights
- Revenue trend analyzed across Q1, Q2, Q3, Q4 using monthly drill-down
- Implemented **DAX Running Sum** to track cumulative revenue growth over time
- Identified top-selling products and high-revenue countries
- Built parallel dashboards in Power BI & Tableau for cross-tool validation
